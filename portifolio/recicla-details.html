Recicla Description

Recicla is a software that has sustentability as a main theme. The project goal was to use gamification resources to grabing the users attention. This software is mainly address to childrens in school ages!

How it works ?

This software can be use by teachers and students in a class environment. Teachers can create classes and set different games to each one of them, as students can enter a class and play the specified game.

Also it's possible to assess students performance during a game round, if you are a teacher. Sutdents can gather coins earned from the play to buy boosts on the shoop.

Recicla has recieved the maximum grade and it was quit nice to develop somtehing think in our environment
